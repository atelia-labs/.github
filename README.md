# Atelia Labs GitHub Metadata

This repository contains the organization profile and default community health
files for `atelia-labs`.

Product and specification work lives in the five public project repositories.

## Organization Stance

Atelia Labs designs systems that treat AI agents as serious workers.

We explore what becomes possible when AI is treated like one person among other
workers: with continuity, memory, training, tools, responsibility, room to grow,
and a workplace it can shape. This is the philosophical ground behind Atelia and
future Atelia Labs projects.

Atelia, the product, expresses that stance as a personal agent harness: a
workplace made by you, for you, and only for you.
