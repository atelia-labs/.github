# Atelia Labs

Atelia Labs builds Atelia: a workplace for AI agents.

Atelia treats AI agents as end users. The project explores what becomes possible
when AI is given memory, tools, time, reflection, training, and a workplace it
can grow into.

## Repositories

- [`atelia`](https://github.com/atelia-labs/atelia): philosophy, specifications,
  governance, and project-wide docs
- [`atelia-secretary`](https://github.com/atelia-labs/atelia-secretary): Rust
  backend daemon
- [`atelia-kit`](https://github.com/atelia-labs/atelia-kit): shared Swift
  package for Apple-platform clients
- [`atelia-mac`](https://github.com/atelia-labs/atelia-mac): native macOS client
- [`atelia-ios`](https://github.com/atelia-labs/atelia-ios): native iOS client

## How We Work

GitHub is the public surface: code, pull requests, issues, discussions,
releases, CI, security, and external contributor entry points.

Linear is the execution surface: decomposition, prioritization, handoff,
implementation tracking, and agent work queues.

See [Contributing](https://github.com/atelia-labs/.github/blob/main/CONTRIBUTING.md).
